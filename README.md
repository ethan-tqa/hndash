# READ ME — HackerNews Dashboard

This is a very casual project, with the goal of helping myself to go through a large amount of HN posts, both current and backlog. What it does, is pull posts from HN that pass certain threshold (don't wanna waste my time on low quality posts), produce a summary using LLM. I can then take a look and decide if I want to read the actual post or linked article.

This project is meant for me only, so not accepting feature requests, pull requests, issues. If you want to make it suitable for your needs, fork it.
